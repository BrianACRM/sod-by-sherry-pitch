# Sod By Sherry Pitch Prototype

A clickable redesign concept for the public Sod By Sherry website.

The prototype keeps the business facts from the current site, then reframes the experience around customer decisions:

- Pick the right sod variety by yard condition.
- Estimate pallet count from square footage.
- Surface credibility from major turf projects.
- Make phone, email, order, pickup, and service-area information easier to find.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This repo includes a GitHub Pages workflow in `.github/workflows/deploy.yml`.
